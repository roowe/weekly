# Notes 目录语言说明

## 语言偏好
在该目录下的所有文件和内容中，**优先使用中文**。

## 使用原则
1. **中文优先**：所有新创建的笔记、总结和分析应使用中文
2. **保留原始引用**：原始标题、作者姓名和技术术语可保留英文原貌
3. **翻译质量**：确保翻译准确，保持原文的技术准确性和核心思想
4. **补充说明**：对复杂概念可添加中文解释或说明

## 文件命名
- 建议使用描述性的中英文混合文件名
- 格式示例：`作者名-主题-日期.md` 或 `topic-summary.md`
- 保持文件名简洁、有意义

## 内容结构
1. **标题**：中文标题（可包含英文原标题作补充）
2. **元数据**：作者、日期、来源等信息
3. **正文**：完整的中文内容
4. **引用**：原始参考文献和链接
5. **备注**：个人思考、疑问和行动计划
